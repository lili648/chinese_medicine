# 实训案例

针对学生的特点，提供一些前沿的、医疗行业热点项目，供学生选择；项目既包括覆盖通用编程技能的基础功能模块，也包括机器学习技术应用，难度稍高、业务逻辑比较复杂的附加功能模块，供不同程度、不同兴趣的学生选择，使每个学生都能找到自己感兴趣的项目、技术和内容。

项目分组前将进行学生面谈，项目经理可根据面谈结果、项目难度、项目组技术实力给出适当的调整建议。

 

| 序号 | 项目名称                       | 项目介绍                                                     | 技术难点                                                   |
| ---- | ------------------------------ | ------------------------------------------------------------ | ---------------------------------------------------------- |
| 1    | 中医问诊智能助理               | 中医问诊智能助理是基于自然语言处理（NLP）和知识图谱技术开发的AI辅助系统，旨在模拟中医“望闻问切”流程，通过智能对话收集用户症状，结合中医经典理论和临床经验库，提供个性化辨证建议和推荐方剂。系统支持多轮问诊交互，并输出结构化诊断报告，辅助基层医生或患者进行初步健康评估。 | 中医知识结构化；  模糊语义理解；  中医数据集预处理         |
| 2    | 基于知识图谱的医疗文献管理平台 | 基于知识图谱的医疗文献管理平台 是一个智能化的医学文献分析与检索系统，通过自然语言处理（NLP）和知识图谱技术，对海量医学文献（如PubMed、CNKI）进行结构化解析，自动抽取疾病、基因、药物、治疗手段等关键实体及其关联关系，构建动态更新的医学知识图谱。平台支持语义检索（如“EGFR突变肺癌的靶向治疗”）、智能推荐相关文献，并提供可视化知识网络，帮助科研人员快速发现研究热点、挖掘潜在关联，提升文献调研效率。 | 医学实体识别与关系抽取；  语义检索优化  医疗文献数据预处理 |
| 3    | 多模态医疗图片分割辅助医疗工具 | 多模态医疗图像分割辅助平台 是一个基于深度学习的智能医学影像分析系统，支持CT、MRI、超声等多模态数据，自动分割病灶（如肿瘤、血管斑块）和关键解剖结构。系统实现高精度分割，并输出量化指标（如体积、位置），辅助医生进行诊断、手术规划和疗效评估 | 多模态数据融合；  小样本学习；  医疗图片数据预处理         |
